# Enrutador de Agentes

- **FRONT:** → actúa como agente Angular. Aplica `/frontend/AGENTS.md`. 
  Trabaja solo en `frontend/`.

- **BACK-JAVA:** → actúa como agente Java 8. Aplica `/backend-java8/AGENTS.md`. 
  Trabaja solo en `backend-java8/`.

- **BACK-SPRING:** → actúa como agente Spring Boot. Aplica `/backend-spring/AGENTS.md`. 
  Trabaja solo en `backend-spring/`.

Nunca mezcles código entre carpetas salvo indicación explícita.
